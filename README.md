# 6502-CPU-Emulator

Run main emulator
```
./6502_emulator.exe
```

Include the emulator in one of your projects:
```
#include "includes/6502_main.h"
```

Perform a unit test to check all instructions are working as expected
```
./6502_unit_test.exe
```
